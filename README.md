Description

- Built RESTful API using Laravel 10 and MySQL.
- Implemented API Specs using OpenAPI for Users, Contacts, Addresses.
- Implemented Eloquent ORM for querying the database and Laravel Migration for database schema.
- CRUD Feature for all entities (Users, Contacts, Addresses).
- Implemented One to Many Relationship between User to Contact and Contact to Address. 
- Implemented dummy data with Laravel Seeder.
- Implemented simple Registration, Login and Logout features using Laravel Middleware.
- Implemented Laravel Form Validation and custom json exception handling.
- Using Laravel Resource for data transform json for API Response structure.
- Implemented Simple Feature Tests for all entities.
- Implemented Clean Architecture Code and Dependency Injection.
